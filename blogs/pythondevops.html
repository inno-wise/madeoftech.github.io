Python Libraries & DevOps Cheat Sheet

1. AI and Machine Learning (scikit-learn)

Scikit-learn is a machine learning library that provides simple and efficient tools for data mining and data analysis. Below are some common operations:

Importing scikit-learn libraries

from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

Basic Machine Learning Example (Random Forest)

# Example: Random Forest Classifier

# Load dataset (e.g., Iris dataset)
from sklearn.datasets import load_iris
data = load_iris()
X = data.data
y = data.target

# Split dataset into train and test sets
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Create and train the Random Forest Classifier
model = RandomForestClassifier(n_estimators=100)
model.fit(X_train, y_train)

# Predict and calculate accuracy
y_pred = model.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
print(f"Accuracy: {accuracy * 100:.2f}%")

2. Django Web Framework

Django is a high-level Python web framework that allows you to build secure and scalable web applications quickly.

Basic Django Project Setup

# Install Django
pip install django

# Create a new project
django-admin startproject myproject

# Create a new app within the project
cd myproject
python manage.py startapp myapp

# Running the development server
python manage.py runserver

Basic Django View

# In views.py
from django.http import HttpResponse

def home(request):
    return HttpResponse("Hello, Django!")

Adding URL to urls.py

# In urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]

3. NumPy (Numerical Computing)

NumPy is a powerful numerical computing library used for array manipulation, mathematical operations, and scientific computing.

Creating and Manipulating Arrays

import numpy as np

# Create an array
arr = np.array([1, 2, 3, 4, 5])

# Perform basic operations
print(arr + 5)   # Array addition
print(arr * 2)   # Array multiplication

# Reshape an array
arr_reshaped = arr.reshape(1, 5)
print(arr_reshaped)

Matrix Operations

# Create two matrices
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix multiplication (dot product)
C = np.dot(A, B)
print(C)

4. Python DevOps Cheat Sheet

DevOps is a combination of development and operations practices aimed at automating and improving the software delivery process. Below are some key tools and practices in Python DevOps:

Docker: Creating a Dockerfile for a Python Application

# Dockerfile for a Python application

# Use official Python image
FROM python:3.9-slim

# Set the working directory inside the container
WORKDIR /app

# Copy requirements file and install dependencies
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Copy the application files
COPY . .

# Expose port 8000
EXPOSE 8000

# Command to run the application
CMD ["python", "app.py"]

Running Docker Containers

# Build the Docker image
docker build -t my-python-app .

# Run the Docker container
docker run -p 8000:8000 my-python-app

Kubernetes: Deploying a Python App with Kubernetes

# kubernetes-deployment.yaml

apiVersion: apps/v1
kind: Deployment
metadata:
  name: python-app
spec:
  replicas: 3
  selector:
    matchLabels:
      app: python-app
  template:
    metadata:
      labels:
        app: python-app
    spec:
      containers:
      - name: python-app
        image: my-python-app:latest
        ports:
        - containerPort: 8000

CI/CD with GitHub Actions

# .github/workflows/ci.yml

name: Python CI

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v2

      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: '3.9'

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt

      - name: Run tests
        run: |
          pytest

Deploying with Ansible

# Ansible playbook example

- name: Deploy Python Application
  hosts: servers
  tasks:
    - name: Update apt cache
      apt: update_cache=yes

    - name: Install Python and dependencies
      apt:
        name:
          - python3
          - python3-pip
      state: present

    - name: Install requirements
      pip:
        name: "{{ item }}"
        state: present
      with_items:
        - flask
        - gunicorn

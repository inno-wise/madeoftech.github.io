Windows PowerShell

PowerShell Cheat Sheet

Basic PowerShell Commands

Get-Help

Shows help for a command.

Get-Help  # Shows help for a command

Get-Command

Find a command by name.

Get-Command  # Find a command by name

Get-Process

Lists running processes on the system.

Get-Process # Lists running processes on the system

Advanced PowerShell Commands

Get-EventLog

Get events from the event log. Useful for checking system events and issues.

Get-EventLog -LogName Application -Newest 10 # Get the 10 most recent application logs

Set-ExecutionPolicy

Change the script execution policy on your machine. Useful for allowing or restricting script execution.

Set-ExecutionPolicy RemoteSigned # Allows running scripts created locally, but blocks unsigned scripts from the internet

Get-ItemProperty

Gets the properties of a specified item, such as registry entries or file properties.

Get-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer" # Get properties from the registry

Get-WmiObject

Get management information from local or remote machines, such as hardware information or system configurations.

Get-WmiObject -Class Win32_OperatingSystem # Get OS information

Invoke-Command

Run commands on remote computers, useful for remote administration.

Invoke-Command -ScriptBlock { Get-Process } -ComputerName Server01 # Run Get-Process on a remote server


Windows CMD

Windows CMD Cheat Sheet

Basic CMD Commands

dir

Lists all files and directories in the current directory.

dir

cd

Changes the current directory.

cd  # Navigate to a directory

cls

Clears the screen in the command prompt.

cls

Advanced CMD Commands

ping

Sends network packets to a specific host to test connectivity.

ping  # Ping a host to test network connectivity

ipconfig

Displays network adapter configuration details like IP address and subnet mask.

ipconfig /all # Display detailed network configuration

tasklist

Lists all running processes on your machine.

tasklist # List all running processes

taskkill

Kills or terminates a running process by name or PID (Process ID).

taskkill /PID  # Kill a process by its PID
taskkill /IM  # Kill a process by its name

netstat

Displays active network connections and listening ports on your computer.

netstat -ano # Show active connections and their associated PIDs

shutdown

Shuts down or restarts the system with a specified timeout and options.

shutdown /s /t 30 # Shut down the system after a 30-second delay
shutdown /r /t 0  # Restart the system immediately


linux CLI

Linux Command Cheat Sheet

Basic Linux Commands

ls

Lists files and directories in the current directory.

ls

cd

Changes the current directory.

cd  # Navigate to a directory

pwd

Prints the current working directory.

pwd

Advanced Linux Commands

grep

Searches for a specific pattern in a file or output.

grep "pattern"  # Search for pattern in file

find

Finds files and directories in a directory hierarchy.

find  -name  # Find a file by name

top

Displays real-time system information, including CPU usage and memory consumption.

top # Display system usage

chmod

Changes the permissions of a file or directory.

chmod 755  # Give read, write, and execute permissions to the owner and read-execute to others

ps

Shows a list of running processes.

ps aux # List all processes running on the system

kill

Terminates a running process by its PID.

kill  # Kill a process by its PID

tar

Used to archive multiple files and directories into a single file (tarball).

tar -cvf .tar   # Create a tarball
tar -xvf .tar # Extract tarball

df

Displays information about disk space usage on the file system.

df -h # Display disk space in human-readable format
